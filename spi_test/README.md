**Output/Input**

* GPIO 7 as SPI0 CE1 (Chip Enable 1)

* GPIO 8 as SPI0 CE0 (Chip Enable 0)

* GPIO 9 as SPI0 MISO

* GPIO 10 as SPI0 MOSI

* GPIO 11 as SPI0 SCLK

* GPIO 47 (ACT LED) as Output

* HDMI as VIDEO Output
